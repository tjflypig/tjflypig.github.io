《曾国藩传》读书笔记——（七）功大不喜，过小必究

发表于 2022-12-26 更新于 2023-07-31 分类于 读书笔记 Waline： 阅读次数：

原本是想一口气更新完的，可是架不住工作太多。家人也来到上海了，时间的分配方面捉襟见肘。年底的工作处理的差不多，恰逢居家办公，也想把这个系列善终一下。

悔生吉

曾大人做事的特点就是，“功虽大而不喜，过虽小而必究”。他常说“悔生吉”，对失败进行深刻的反省，每一次失败都要吃透，才能不白白付出代价。
他亲自指挥的靖港战役失败的教训是惨痛的，虽然有很大一部分原因在于前期的信息有误差，对太平军的兵力估计不足。但是这一仗损失了湘军三分之一的兵力，也让曾大人第一次投水自杀。
此处也让我想起指环王里面的一句台词：“哪些成功的人，在前进的路上也有无数次想放弃，正是因为他们坚持下来了，所以他们才成功了。”
这一仗过后，虽然湘潭大捷，但曾大人还是深刻反省，大量换人，大幅减员，精进兵力，论功行赏，创新制度。

《曾国藩传》读书笔记——（五）行有不得，反求诸己

发表于 2022-05-27 更新于 2023-07-31 分类于 读书笔记 Waline： 阅读次数：

继续来把我的读书笔记更新完，最近上海要解封了，事情也多了起来，所以这个“挺”字，说起来容易，做起来真的是难。

“行有不得，反求诸己”——《孟子》的《离娄章句上》

此“反求诸己”乃真“反思”也

老祖宗的智慧，总是精辟有高度，“行有不得，反求诸己”是对自己的高度要求，我理解为一种思考模式，做不成某些事情，先从自己的角度考虑问题，这又是一件说起来容易做起来难的事情。

结合之前说的日记，曾大人每天都在反思做人做事中自己的不足与缺点，死而后已。才有了他品行的不断提升和为人处世方式的不断优化。

《曾国藩传》读书笔记——（四）“反思”习惯

发表于 2022-05-13 更新于 2023-07-31 分类于 读书笔记 Waline： 阅读次数：

“反思”非反思

这里的“反思”并不是，自省反思的反思，而是反向思考。曾国藩在创建湘军的时候，开始想的并不是我要把湘军打造的怎么样，有多强的战斗力，而是分析了失败的案例——“正规军为啥没有战斗力？”

一个很朴实无华的思想，不过是别人犯过的错误，别人做的不对的，不去做，反着来，是不是就有可能成功呢？

所以，先深入分析前人失败的经验，然后“反着思考”，这就是曾大人开始对湘军主要政策设计的思路。
